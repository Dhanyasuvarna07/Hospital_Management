# 🏥 Hospital Management System

A simple full-stack Hospital Management System built using **Node.js, Express, MySQL, HTML, CSS, and JavaScript**.  
It manages patients, appointments, billing, and treatments with database automation using **MySQL triggers**.

---

## 🚀 Features

### 👨‍⚕️ Patient Management
- Add new patients
- Update patient details
- Delete patients
- View all patients
- Track patient visit count automatically

---

### 📅 Appointment Management
- Add appointments for patients
- Automatically updates visit count using MySQL triggers

---

### 💰 Billing System
- Generate bills for patients
- View all bills with patient names

---

### 🏥 Database Modules
- Patients
- Appointments
- Bills
- Treatments
- Doctors
- Departments

---

## 🧠 Database Design (MySQL)

### 📌 Tables Used
- `patients`
- `appointments`
- `bills`
- `treatments`
- `doctors`
- `departments`

---

## ⚡ MySQL Triggers Used

### ✔ Auto Update Visit Count
When a new appointment is inserted:

- `VisitCount = VisitCount + 1`

This ensures **no backend logic is required for visit tracking**.

---

## 🛠️ Tech Stack

| Layer | Technology |
|------|------------|
| Frontend | HTML, CSS, JavaScript |
| Backend | Node.js, Express.js |
| Database | MySQL |
| API | REST API |
| Styling | Custom Black & White Theme |
